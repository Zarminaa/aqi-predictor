# Pearls AQI Predictor

An end-to-end MLOps project that predicts Air Quality Index (AQI) for the next 3 days.

## Features

- Historical data collection
- Feature Engineering
- Feature Store (Hopsworks)
- Model Registry
- Automated Training
- Streamlit Dashboard
- SHAP Explainability
- GitHub Actions CI/CD

## Tech Stack

- Python
- Scikit-learn
- Pytorch
- Hopsworks
- GitHub Actions
- Streamlit
- Flask
- OpenWeather API
- SHAP
